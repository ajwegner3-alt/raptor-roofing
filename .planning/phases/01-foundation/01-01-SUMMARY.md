## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] node_modules copy conflict during sibling temp dir merge**

- **Found during:** Task 1
- **Issue:** The first `cp -rn` command ran as a background job and began copying node_modules (~358 packages, thousands of files). When Task 2 tried `npm install`, npm errored on a locked/non-empty zod directory mid-copy.
- **Fix:** Removed the incomplete node_modules directory and ran `npm install` fresh from the committed package.json. This produced a clean, verified install.
- **Files modified:** node_modules (removed + reinstalled), package-lock.json (regenerated)
- **Commit:** 611dffd (package-lock.json update included)

**2. [Rule 2 - Missing] lint script was "eslint" (no target path)**

- **Found during:** Task 2
- **Issue:** The scaffolded package.json had `"lint": "eslint"` without a path argument, which would cause eslint to do nothing when invoked.
- **Fix:** Patched to `"eslint ."` per Next.js 16 guidance and plan spec.
- **Commit:** 611dffd

**3. [Rule 2 - Missing] package name was left as "raptor-roofing-scaffold-tmp"**

- **Found during:** Task 2
- **Issue:** Scaffold temp dir name leaked into package.json `name` field.
- **Fix:** Updated to `"raptor-roofing"`.
- **Commit:** 611dffd
