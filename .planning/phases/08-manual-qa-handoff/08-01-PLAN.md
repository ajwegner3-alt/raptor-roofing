THREE surgical fixes:

1. **UrgencyBar.tsx lint error (line ~17):** Replace the `useEffect` that reads sessionStorage and calls `setDismissed(true)` with a lazy state initializer. Change:
   ```typescript
   const [dismissed, setDismissed] = useState(false);
   useEffect(() => {
     if (sessionStorage.getItem(SESSION_KEY) === "1") {
       setDismissed(true);
       return;
     }
     // ... rest of effect
   }, []);
   ```
   To: Initialize `dismissed` lazily: `useState(() => typeof window !== 'undefined' && sessionStorage?.getItem(SESSION_KEY) === '1')`. Then adjust the remaining effect to only handle the scroll timer logic (not the sessionStorage check). Be careful with SSR — `typeof window` guard is needed.

Move these state resets into the close handler function that sets `open` to false, rather than reacting to the `open` state change in an effect. Read the component first to understand the close handler pattern, then restructure.
